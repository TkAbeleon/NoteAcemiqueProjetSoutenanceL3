# Etude de l'existant:
## Gestion des matériels informatiques
Dans une société, on souhaite gérer les matériels informatiques de tous les services dans une application informatique.
En cas de pannes des matériels informatiques, les utilisateurs peuvent faire une demande d'intervention à l’agent informatique
L'agent informatique a pour rôle de traiter les incidents et aussi de faire affectation des matériels en cas d’insuffisance dans les autres services. **Tout cela** est saisi dans un **formulaire** d'une **application** avec laquelle l'agent informatique doit s'authentifier.
De plus l'agent informatique est sous la direction d'un chef informatique
1. Établir le diagramme de cas d’utilisation de cette situation, on prendra comme nom du système:***Gestion des matériels informatiques***
2. Faire les descriptions textuelles de chaque cas d’utilisation
3. A partir de ces description textuelles, réaliser les digrammes de séquence correspondante
4. Réaliser les diagrammes d'activités correspondants à  chaque cas d’utilisation
5. Élaborer le diagramme de classe du système.
